# Microsoft Entra ID Overview

## Definition
Microsoft Entra ID (Azure AD) is a cloud-based Identity and Access Management (IAM) system that allows you to manage:
- Users, groups, and devices
- Applications and service principals
- Roles and access
- Authentication and security policies

## Key Capabilities
| Feature | Purpose |
|---------|---------|
| User Management | Add/delete users (cloud, guest, on-prem, hybrid) |
| Group Management | Security & Microsoft 365 groups, assigned & dynamic members |
| Role Management | Assign Entra roles (Global Admin, User Admin, etc.) |
| Applications | Enterprise applications & App Registrations |
| Identity Protection | Detect risky sign-ins/users |
| Conditional Access | Enforce MFA, compliance, device-based policies |
| Integration | Intune, on-prem AD, apps via Entra Connect |
| License Management | Assign licenses to users or groups (dynamic) |

## Interview Line
"Microsoft Entra ID manages cloud identities, secures access, and integrates with applications, devices, and Azure resources."
